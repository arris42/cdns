# Temporary Files

Download Anythings...<br/>
Download Everywhere...

**How To**
- Using curl<br/>
`curl https://raw.githubusercontent.com/<username>/<repo>/master/<filename> --output <filename>`

- Using wget<br/>
`wget -O <filename> https://raw.githubusercontent.com/<username>/<repo>/master/<filename>`
&nbsp;

**Examples**
- `curl https://raw.githubusercontent.com/foo/myrepo/master/unzip.exe --output unzip.exe`
- `wget -O unzip.exe https://raw.githubusercontent.com/foo/myrepo/master/unzip.exe`
